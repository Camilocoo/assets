# Replit Exercises API

#### Get single syllabus
```
GET: replit/cohort/{slug}
```

#### Get all replit from particular cohort
```
GET: replit/cohort/{cohort_slug}
```

#### Save all replits from a particular cohort
```
POST: replit/cohort/{cohort_slug}
```
