# Lessons API

Get all lessons
```
GET: /lesson/all/v2

PARAMS:
- status (string): Filter the lessons by status ['draft, 'publish']
```

Get all lessons from the old wordpress
```
GET: /lesson/all

PARAMS:
- status (string): Filter the lessons by status ['draft, 'publish']
```
