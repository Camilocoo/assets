# Static Images API

#### Get all images
```
GET: static/image/all
```

#### Create single image
```
{
    "tags": "html, js, jsx",
    "description": "Cheat sheet explaining css",
    "category": "learn-html",
    "url": "https://ucarecdn.com/245d1b5e-19f1-482d-8c77-b2debac894bb/",
    "sizes": []
}
```
